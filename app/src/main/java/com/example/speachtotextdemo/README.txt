The main feature of this app is Speech to text
This project architecture is MVVM & the programing language is Kotlin.
Skill set used- CoRoutine,Retrofit,Dependency Injection(Kodin),Mockito
Unit test->Junit & Mockito
---------------Features----------
1.Text is listed below is coming from backend API.
2.Speak button used to take the user Spoken text,
3.When user spoken word is matches with listed word then the row get high light & the value get incremented.
4.The list is Always be in Descending order w.r.t counts
